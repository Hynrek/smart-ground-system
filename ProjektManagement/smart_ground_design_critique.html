Priority recommendations
1
Press-and-hold for the fire button
Show a circular progress ring — trigger at 600ms. Cancel on release. This prevents accidental hardware triggers on a touch tablet and communicates intent.
2
Add status color to Range cards
A colored left border (green / amber / red) based on device health lets the Standwart triage problems without opening each range individually.
3
Increase touch targets to 44×44px
Edit, remote, and unassign icon buttons are ~28px. Add padding. This is a mandatory fix for the tablet context.
4
Label the Remote button
Always show "Remote" text beside the icon on range cards. The program icon alone is not recognizable to new users.
5
Fix contrast failures + add aria-labels
#a0aec0 on white fails WCAG AA — shift secondary text to #718096 minimum. Add aria-label to all icon buttons and StatusDot for screen readers.
