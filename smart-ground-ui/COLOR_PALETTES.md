**Current default - what users see today**

```

Sidebar:           #1a1a2e (Dark Navy)

Text Secondary:    #4a5568 (Medium Gray)
Success:           #48bb78 (Green)
Warning:           #ed8936 (Orange)
Error:             #f56565 (Red)
```

**Strengths:**
- ✅ Cool, modern, tech-forward feeling
- ✅ High contrast sidebar (dark bg, light text)
- ✅ Good for device control (cyan feels "activated")
- ✅ Familiar to users (used currently)

**Weaknesses:**
- ⚠️ Cyan might be too saturated for long viewing
- ⚠️ Navy sidebar can feel dark/heavy
- ⚠️ May not feel approachable to non-tech users

**Best For:** Tech-savvy operators, industrial settings, shooter community

## 📱 Responsive Considerations

All palettes are designed to work across:
- Desktop (dark backgrounds, high contrast)
- Tablet (scaling considerations)
- Mobile (touch-friendly, readable)
- Outdoor (bright sunlight visibility)
- Indoor (dark-adapted eyes)

**Gaming palette** performs best in low-light conditions.  
**Professional and Modern** palettes perform best in mixed lighting.  
**Minimal palette** performs best in bright/outdoor conditions.

## 🔧 Implementation Notes

### CSS Variables Available
Each palette provides:
- `--color-primary` (main accent color)
- `--color-primary-dark` (darker variant for hover)
- `--color-primary-light` (lighter variant for bg)
- `--color-sidebar-bg` (sidebar background)
- `--color-sidebar-text` (sidebar text)
- `--color-bg-main` (main background)
- `--color-bg-card` (card background)
- `--color-text-*` (text colors)
- `--color-success/warning/error` (semantic colors)

### Migration Path
To switch palettes in code:
1. Set `data-palette` attribute on `<html>` element
2. Components automatically pick up new CSS variables
3. No code changes needed in Vue components

```vue
<!-- In PaletteSwitcher.vue -->
document.documentElement.setAttribute('data-palette', 'professional');
```

### Color Accessibility
All palettes meet **WCAG AA** standards (4.5:1 contrast ratio).  
**Professional, Modern, and Minimal** meet **WCAG AAA** (7:1) for most text.
